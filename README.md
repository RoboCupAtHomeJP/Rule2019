# オブジェクト
Open Platform リーグでは，Known Object・Unknown Object共にYCB Objectとは別に運営で用意します．  
Domestic Stadard Platform リーグでは全競技において，Known ObjectとしてYCB Objectを使用し，
Unknown ObjectはYCB Objectとは別に運営が用意したオブジェクトを使用します．
